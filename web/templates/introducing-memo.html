Introducing Memo

We're announcing the launch of Memo, an on-chain social network built on Bitcoin Cash.

What is Memo? Why was it built? How does it work? This post should answer these questions.

Memo is both a protocol and a front-end application. The protocol works by writing OP_RETURN transactions to the Bitcoin Cash blockchain. Client applications read these transactions to see the activity on the network.

With current social networks users do not control their data and can have their accounts removed without reason. The blockchain is more than an uncensorable way to send money, it's an uncensorable way to store data. Memo is an experiment leveraging the blockchain's data storing capabilities.

Each action is stored on the blockchain in a transaction - whether it's posting a memo, following someone or liking a post. You can also include a tip (it is on-chain after all). There are additional actions that are intended to be added later.

The Memo website is a front-end application built on top of this protocol. There currently is not an open-source client, but one is planned to be released later.
